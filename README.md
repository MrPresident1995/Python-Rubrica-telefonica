Si desidera gestire una rubrica telefonica. Si vuole gestire un certo numero di
nominativi, memorizzando nome, cognome, numero telefonico ed email.
L'utente può:
1. aggiungere un nominativo
2. cancellare un nominativo
3. cercare uno o più nominativi, solo per cognome.
4. modificare i dati di un dato nominativo
5. stampare l’elenco dei nominativi
